## About

The FASH E-commerce Website offers a comprehensive online shopping experience, providing a wide range of products for various categories. This platform aims to provide customers with an easy-to-use interface and a seamless shopping journey.

## Features

- Explore a diverse collection of products.
- Categorized products for easy navigation.
- User-friendly shopping cart functionality.
- Secure checkout process.
- Responsive design for optimal viewing across devices.

## Technologies Used

- HTML5, CSS3, JavaScript
- UNSPLASHED API 
- Netlify (Deployment)

## Getting Started

## Usage

- Browse through product categories.
- View product details and images.
- Add products to your cart.
- Proceed to the secure checkout process.
